Javascript Funções

Funções declarativas

Também podem ser conhecidas por funções nominais, por levarem um nome. Esta técnica geralmente é muito usada por programadores js iniciantes. No exemplo abaixo o nome da função é getName.

function getName() {

        return "Rodrigo";

}

console.log(getName());

Expressões de funções

Também pode ser chamadas de funções anônimas. Estas funções funciona quando atribuída a uma variável. Esta técnica é bastante usada por programadores mais experts. Porém, a diferença entre elas dependem muito do contexto. Veja um exemplo abaixo.

var getAge = function() {

        return 33;

}

console.log(getAge());
