Ferramenta para substituir outras do mercado…

Ele roda sobre o node.js [js do lado do servidor]. Node.js foi feito pelo Google, uma engime V8, toda escrita em C++

É necessário instalar o NPM [node package module]:  gerenciador de pacotes q rodam em cima do node.js. O maluco aqui gerencia os  módulos, uma espécie do plugin.

Touch GruntFile.js

module.exports = function(grunt) {

        grunt.initConfig ({

                //add uma task

                copy: {

                        all: {

        expend: true,

        cwd: “dev”,

        src: “**”,

        dest: “dist”

}

}

})

}

Quem copia esta bagaça?

grunt.loadNpmTasks(“grunt-contrib-copy”);

>- terminal

npm install grunt-contrib-copy
